
# 折叠面板

## API

### collapse

| 参数      | 说明             | 类型      | 默认值  |
|----------|------------------|----------|--------|
| width    | 指定面板的宽度     |  百分比宽度, 及**px  |  100% |

### RdkPanel
| 参数      | 说明             | 类型      | 默认值  |
|----------|------------------|----------|--------|
| isActive | 是否默认打开 |  boolean  | false |
| header | 折叠面板的标题 | string |  - |
