
# 输入框

## API

### Input

| 参数      | 说明             | 类型      | 默认值  |
|----------|------------------|----------|--------|
| width | 控件宽度 | string: number/px/% | 300px |
| height | 控件高度 | string: number/px/% | 28px |
| value | input的表单值 `双绑` | string 或者 number | - |
| valueChange | value变化的回调 | EventEmitter | - |
| clearable | 是否支持清空 | boolean | true |
