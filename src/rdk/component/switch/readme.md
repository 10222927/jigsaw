
# 开关

## API

### switch

| 参数    |   说明         | 类型    | 默认值 |
|--------|--------------| -------| -----|
| checked  | 是否选中      | boolean  |  false |
| checkedChildren | 选中显示的内容| string | - |
| unCheckedChildren | 非选中的内容 | string | -|
|disabled   | 是否禁用      |boolean | false|
| change    | 变化时事件     |Function(c:checked)| - |
